Folder Structure

This template is build in bootstrap 2.3.3 however some of the core classes of bootstrap have been changed to better improve the performance and look and feel of the theme.

List of the folders and special file mentions as follows:

  • css: where the CSS files are stored
  • fonts: additional custom fonts including woocommerce, fontawesome and rating stars
  • images: where the images are stored
  • isotope_gallery: isotope plugin
  • js: where the javascript files are stored
  • layerslider: Parallax slider

Special mention: Responsiveness

You can turn on and off responsiveness by deleting the or commenting out bootstrap-responsive.css from style.css

Contact

Please note: The form needs to be uploaded to your website/server for it to work. If you run the form in your PC/Mac the validation will work but you cannot submit any messages.
To change the email address where you receive the messages from the Contact Us section, open the file named ajaxSubmit.php. Once opened you will find the following code:

$receiver = "email@gmail.com" ;

Simply substitute "email@gmail.com" with your email address and save. That's it!
